## Active Project

### 🚀 Dual Encoder for Semantic Search

**Status:** 🔵 Active - In Development

**Description:**
A neural architecture designed for efficient semantic similarity comparison between queries and documents. This project implements a dual encoder model with in-batch negative sampling for information retrieval tasks.

**Technologies:** PyTorch, Transformers, FAISS, FastAPI

**Features:**
- Query and document encoding into shared embedding space
- Cosine similarity scoring for relevance ranking
- Scalable vector search with FAISS indexing
- REST API for deployment

**Current Milestones:**
- ✅ Model architecture design
- ✅ Training pipeline implementation
- 🔄 Hyperparameter optimization
- ⏳ Production deployment
